Project UART_CMSIS_IT

Проект демонстрирует UART-echo 
Используется UART1 подлюченный через конвертер UART-TTL к COM порту компьютера 
Взаимодействие происходит через программу терминал. (Putty, CoolTerm, или аналогичные)

Pinned in stm32F103C8T6:
RX -> A10
TX -> A9

Не забывайте, что подключение по интерфейсу UART производится следующим образом:
 DEV1		DEV2
 RX		<->	TX
 TX		<->	RX
 GND	<-> GND
